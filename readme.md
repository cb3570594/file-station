# 文件管理器

## Install

### 将项目克隆到需要管理的文件目录所在的文件夹下

执行

`yarn install`

or

`npm install`

## Start

`yarn start`

or

`npm start`

访问 [http://localhost:3000]

## Config

### baseDir: 需要管理的目录名称(如./files 目录), readAll: 是否递归获取所有文件, outputPath: 将读取生成的 files 对象输出的 json 文件路径, port: 默认监听的端口号

## Apis

### delete, upload, addDir, removeDir
